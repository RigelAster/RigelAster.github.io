Cg Standard Library

因为Unity的ShaderLab中包含了cg，所以cg标准库中的函数可以直接在Unity Shader中使用

网址 NVIDIA Developer Cg Standard Library

step

Parameters

a Scalar or vector reference value

x Scalar or vector

Description

return one for each component of x that is greater than or equal to the corresponding component in the reference vector a, and zero otherwise

smoothstep

Parameters

a Scalar or vector minimum reference value

b Scalar or vector minimum reference value

x Scalar or vector

Decription

Interpolates smoothly from 0 to 1 based on x compared to a and b

1. Returns 0 if <u>x < a < b</u> or <u>x > a > b</u>
1. Returns 1 if <u>x < b < a</u> or <u>x > b > a</u>
1. Returns a value in the range[0,1] for the domain [a,b]

The slope of smoothstep(a, b, a) and smoothstep(a, b, b) is zero.

For vectors, the return vector contains the smooth interpolation of each element of the vector x

ddx ddy

Parameters

a Vector or scalar of which to approximate the partial derivative with respect to window-space X.

Description

Returns approximate partial derivative of a with respect to the window-space (horizontal) x coordinate.For vectors, the returned vector contains the approximate partial derivative of each element of the input vector.

这个函数单看文档有些难以理解，贴一个方便理解的博客：https://blog.csdn.net/yinhun2012/article/details/109393911

Unity Built-in shader include files

Unity提供了很多内置的工具函数来让编写Shader变得更加简化和容易。这些工具函数都被声明在.cginc文件中，可以在这里查看Unity内置的文件： Built-in shader include files

.cginc是可以被Shader引用的文件，Unity内置如下文件：

  • HLSLSupport.cginc - ( automatically included ) Helper macros and definitions for cross-platform shader compilation
  • UnityShaderVariables.cginc - ( automatically included ) Commonly used global variables.
  • UnityCG.cginc - commonly used helper function
  • AutoLight.cginc - lighting & shadowing functionality, e.g. surface shaders use this file internally
  • Lighting.cginc - standard surface shader(automatically included by surface shader) lighting models
  • TerrainEngine.cginc - helper functions for Terrain & Vegetation shaders

UnityCG.cginc

官方文档：Functions declared in UnityCG.cginc

UnityCG.cginc文件还提供了 Screen-space helper functions，这些函数可以帮助我们计算出屏幕空间坐标，方便对screen-space texture进行采样。其返回float4，方便后续采样时做齐次除法(xy/w)。

除此之外，Unity还提供了用于前向渲染的有关灯光相关的工具函数，请查阅官方文档。

Data structures

UnityCG.cginc还封装了一些包含着色器语义结构体，可以直接使用而不用自主声明，直接用于顶点着色器的数据输入：

  • appdata_base: position, normal and one texture coordinate
  • appdata_tan: position, tangent, normal and one texture coordinate
  • appdata_full:position, tangent, normal, four texture coordinates and color
  • appdata_img: position, one texture coordinate
